#### Getting Started with Prompts

1. Browse the `prompts/` folder to find the category that matches your task
2. Open the relevant prompt template file
3. Copy the template and fill in the placeholders with your specific details
4. Paste into your AI assistant (ChatGPT, Claude, Copilot Chat, etc.)
5. Iterate and refine based on the AI's response
